# PowerPoint Slides - Source Materials

This directory contains the original PowerPoint slides and materials that will be converted to interactive Jupyter notebook lectures.

```

├── README.md                    # This file

├── week02_words.pptx

└── ...                         # Additional slide files
```

## 🔄 Conversion Process

Each PowerPoint file in this directory will be converted to an interactive Jupyter notebook in the parent `lectures/` directory:

- **PowerPoint slides** → **Jupyter notebook lectures**
- **Static content** → **Interactive exercises and code demos**
- **Bullet points** → **Hands-on activities**
- **Theoretical concepts** → **Practical implementations**

## 📝 Conversion Guidelines

When converting slides to notebooks:

1. **Preserve Content**: Keep all key concepts and learning objectives
2. **Add Interactivity**: Convert static examples to runnable code cells
3. **Include Exercises**: Transform bullet points into interactive activities
4. **Maintain Structure**: Keep logical flow and section organization
5. **Enhance with Code**: Add practical demonstrations of concepts
6. **Update References**: Include modern NLP tools and libraries

## 🛠️ Tools for Conversion

- **Manual conversion**: Copy content and restructure for interactive format
- **Content extraction**: Use slide text as markdown cells
- **Code examples**: Create new code demonstrations
- **Visual elements**: Recreate important diagrams and figures
