LSTM — Long Short-Term Memory

Solves the vanishing gradient problem with a cell state and three gates

Forget gate ft
Input gate it
Output gate ot
Cell state ct
Hidden state ht
Press Play or use the step buttons to begin.
ft = σ(Wf·[ht−1, xt] + bf)
it = σ(Wi·[ht−1, xt] + bi)
ot = σ(Wo·[ht−1, xt] + bo)
c̃t = tanh(Wc·[ht−1, xt] + bc)
ct = ft ⊙ ct−1 + it ⊙ c̃t
ht = ot ⊙ tanh(ct)

GRU — Gated Recurrent Unit

Simpler than LSTM: merges cell & hidden state, uses only two gates

rt = σ(Wr·[ht−1, xt])

h̃t = tanh(W·[rt ⊙ ht−1, xt])
