Vanilla RNN: The sky is blue

ht = tanh(W·[ht-1, xt] + b). The hidden state accumulates context word by word. In the rolled view, ht loops back as ht-1 on the next timestep.

LSTM Equations

Forget Gate
ft = σ(Wf·[ht-1, xt] + bf)
Input Gate
it = σ(Wi·[ht-1, xt] + bi)
Candidate Cell
c̃t = tanh(Wc·[ht-1, xt] + bc)
Cell State
Ct = ft × Ct-1 + it × c̃t
Output Gate
ot = σ(Wo·[ht-1, xt] + bo)
Hidden State
ht = ot × tanh(Ct)

GRU: it was not that good

GRU Equations

Reset Gate
rt = σ(Wr·[ht-1, xt] + br)
Update Gate
zt = σ(Wz·[ht-1, xt] + bz)
Candidate Hidden
h̃t = tanh(Wh·[rt×ht-1, xt] + bh)
Hidden State
ht = (1 - zt) × ht-1 + zt × h̃t

The update gate zt combines the roles of forget and input gates from LSTM. No separate cell state is needed.
